# NKnife.TDMS

#### 介绍
NKnife.TDMS 是一个基于 .NET Standard 2.0 和 .NET 8 的数据管理系统，旨在提供高效、可靠的数据管理解决方案。

#### 软件架构
软件架构说明：
- 基于 .NET Standard 2.0 和 .NET 8
- 模块化设计，易于扩展和维护
- 支持多种数据库类型

#### 安装教程

1.  下载并解压安装包
2.  运行安装程序并按照提示完成安装
3.  配置数据库连接和其他必要的设置

#### 使用说明

1.  启动应用程序
2.  登录系统并选择所需的功能模块
3.  根据需要进行数据管理操作
